# **App Name**: Content Alchemy

## Core Features:

- Topic Input: Allow users to input blog topics and specific requirements (tone, keywords, target audience).
- AI Content Generation: Use AI to generate a full blog post draft based on the provided topic and requirements; the tool will output in human-like style that avoids AI detection.
- Integrated Editor: Provide a built-in editor to refine and customize the AI-generated content with formatting options.
- One-Click Export: Enable users to export the finalized content in various formats (.txt, .docx, .html).
- AdSense Compliance: Incorporate AdSense guidelines during content creation to ensure the generated posts are ad-friendly.
- AI Suggestion: Let AI provides headline options, paragraph re-writes and featured image suggestions for improvement.

## Style Guidelines:

- Primary color: Vibrant purple (#9400D3) for a creative and magical feel.
- Background color: Light purple (#E6E6FA) a light and youthful color, desaturated
- Accent color: Electric blue (#7DF9FF) to highlight the most important interactions.
- Body and headline font: 'Belleza' (sans-serif) for headings; 'Alegreya' (serif) for body.
- Use hand-drawn icons related to writing, creativity, and magic to create a beautiful UI.
- Design each screen with smooth gradient backgrounds, ensuring a visually stunning and cohesive look.
- Incorporate animated transitions to content loading, providing a fluid and engaging user experience.
